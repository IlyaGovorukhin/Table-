Задание:
Сделать искрографик и оформить в виде jQuery-библиотеки с дружелюбным АПИ. Задание выполняется после выполнения задания на вёрстку.


Теория:
https://ru.wikipedia.org/wiki/%D0%98%D1%81%D0%BA%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA

В данном случае, искрографик отражает рейтинг пиццерии за 12 недель. Каждый столбик - рейтинг за 1 неделю. Рейтинг лежит в интервале [0; 100] единиц.


Критерии проверки:
1. Стили должны быть редактируемы (с помощью CSS), без изменений в JS файлах. Классы могут быть фиксированы;
2. Вёрстка должна быть сделана по БЭМ (https://ru.bem.info/methodology/naming-convention/);
3. Должно настраиваться количество столбиков;
4. При наведении на столбик - он выделяется и генерируется событие выделения столбика, в котором есть информация о столбике (картинка 2);
5. Вёрстка должна быть сделана с использованием простых html4-элементов (не использовать canvas!);
6. По-умолчанию выделен крайний правый столбик искрографика (картинка 1);
7. Должно быть оформлено в виде библиотеки с дружелюбным АПИ


Дополнения:
1. Подключать jQuery следует с Google CDN;
2. В папке с решением должно быть 3 файла - js и css и html (могут быть так же исходники ts и scss);
3. Дополнительный плюс за использование TypeScript (в этом случае приложить исходник ts и скомпилированный js);
4. Дополнительный плюс за использование SCSS (в этом случае приложить исходник scss и скомпилированный css);
5. Пример должен запускаться без дополнительных манипуляций.
